How to Use / Test a Node in Your Wallet

  1. Open your Monero GUI or Feather Wallet.
  2. Go to **Settings** → **Node** (or Remote Node).
  3. Select "Remote node" and enter one from above (e.g. node.moneroworld.com:18089).
  4. Click "Connect" or save — watch the sync status in the bottom bar.
  5. If it fails (red status), try another node from the list.
Privacy Tip: Remote nodes see your IP and wallet queries. For full privacy, run your own node (see "Download Node" button on dashboard). Use Tor/I2P for remote connections.
